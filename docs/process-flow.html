Functional Consultant Resource Matcher

The MVP connects a Streamlit chat surface to a FastAPI backend over Server-Sent Events. LangGraph orchestrates the workflow, while Excel, Tavily, OpenRouter, and semantic matching provide the lightweight data and AI layer.

1. Streamlit Chat User submits a project or product problem statement.
2. FastAPI SSE UI calls /v1/pipeline/stream and receives progress events.
3. LangGraph Loads resources, searches context, computes strength, and scores fit.
4. Task Matrix Maps each workstream to primary/support resources.
5. PM Report Generates a downloadable task-based staffing report.
Streamlit -> FastAPI -> LangGraph -> Report -> Streamlit

Frontend to Backend

  • Streamlit renders chat bubbles, status updates, and download button.
  • HttpClient consumes SSE progress, step, complete, and error events.
  • Console logs mirror UI progress for debugging and traceability.

Backend Pipeline

  • Excel stores users, skills, levels, and user-skill mappings.
  • Tavily and OpenRouter enrich the problem statement in live mode.
  • Semantic matching computes skill strength and task-level fit.
  • python-docx creates the final Product Manager report.
